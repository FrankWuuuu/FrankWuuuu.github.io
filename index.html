TFT Web Scraper & Web App

Last summer, I built a web application that summarized the best strategies for the online game TeamFight Tactics (TFT). I used Selenium as a web scraper to collect data from various TFT strategy blog websites, and processed the data into one summarizing TierList. Then, using Django and React, I built a web application to present these findings.

This project was my first time learning how to use Selenium, as well as an opportunity to get more comfortable with React and Django.
